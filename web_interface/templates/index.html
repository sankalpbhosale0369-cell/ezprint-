EzPrint

Hybrid Printing System

Welcome to EzPrint

EzPrint is a hybrid printing system that allows customers to upload and print documents remotely through shopkeeper-managed printers.

Features:

  • Upload PDF, DOCX, and image files
  • Customize print settings (page range, copies, size, orientation)
  • Preview documents before printing
  • Real-time print job tracking
  • Secure file handling

How to Use:

  1. Scan the shop's QR code to access their upload page
  2. Upload your document
  3. Customize print settings
  4. Preview your document
  5. Confirm and print
